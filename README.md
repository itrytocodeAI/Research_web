# DeepResearch AI Toolkit

An AI-powered web platform for conducting in-depth research analysis on any topic.

## Features

- AI-powered research and gap identification
- Hypothesis formulation and implementation planning
- Word document and Markdown report generation
- Google Drive integration

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Blink SDK

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```
